02 — How It Works
The Methodology
STEP 01

Benchmark Collection

Raw FPS data sourced from Tom's Hardware GPU Hierarchy at 1440p Ultra settings — the same test suite, the same games, every GPU.

STEP 02

Cross-Generation Normalization

Older benchmarks used a lighter test suite. We compute a scale factor (~0.703) from 20+ GPUs that appear in both suites, then normalize legacy scores to modern equivalents.

STEP 03

Live Pricing

Prices pulled from Newegg monthly — lowest available non-OC model from a reputable AIB, new condition only. No gray market. No cherry-picked sales.

STEP 04

Cost Per FPS

Simple division: price ÷ normalized FPS. Lower is better. No black-box weighting — the number means exactly what it says.

Fully open source

Pipeline, data, and methodology — all public on GitHub. See exactly how the numbers are made.
